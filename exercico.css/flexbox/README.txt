## Css FlexBox

//display flex 


//flex direction
#row = linha. deixando os elementos um ao lado do outro.
#column = coluna. deixando os elementos um abaixo do outro.
#row-reverse = um ao lado do outro na ordem contraria.
#column-reverse = um elemento abaixo do outrona ordem contraria.
//flex wrap
# no wrap = mesmo que nao declarar nada.
# wrap-reverse = os elementos acima dele vão para baixo quando acontecer a quebra de linha.
# wrap = ele quebra o elementopara um aproxima linha quando o parametro definido não é alcançado. 
ex = um lorem não tem espaço para um linha onde ja tem outros elementos, ele coloca esse lorem em forma de column e pula para a linha de baixo onde tem espaço para ele inteiro.
->flex flow = short hand de flex wrap e flex direction.
