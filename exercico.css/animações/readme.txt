## Animações com css
utilizamos os @keyframes para declarar uma animação, utilizamos para declarar os parametros e depois declaramos na nossa aplicação para ser utilizada no objeto que preparamos.

# @keyframes 
Dentro dos keyframes podemos declarar tudo que será transformado pela animação e seu tempo, poe exemplo podemos fazer uma transição entre três cores ate chegar na cor final da transição, dentre outras funcionalidades.
nesse caso declaramos em .loading-spinner com a propriedade animation. nela passaremos como nossa animação ira ser representada na tela, tempo de animação, velocidade da animação e quantas vezes será repetida a animação.

# a propriedade animation pode ser declarada de forma individual sem o uso de short-hands.
